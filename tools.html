Entornos de Desarrollo

Visual Studio

Microsoft Visual Studio es un entorno de desarrollo integrado para Windows, Linux y macOS.

Apache Netbeans

NetBeans es un entorno de desarrollo integrado libre, hecho principalmente para el lenguaje de programación Java.

Aptana Studio

Aptana Studio es un entorno de desarrollo integrado de software libre basado en eclipse y desarrollado por Aptana, Inc.

Programación web

Apache

Servidor web HTTP de código abierto, para plataformas Unix, Microsoft Windows, Macintosh y otras.

MySQL

Sistema de gestión de bases de datos relacional desarrollado bajo licencia dual: Licencia pública general/comercial.

Php

Hypertext Preprocessor, es un lenguaje de programación de propósito general de código del lado del servidor.

Diseño gráfico

Photoshop

Editor de gráficos rasterizados desarrollado por Adobe Systems Incorporated. Usado principalmente para el retoque de fotografías y gráficos.

Adobe Illustrator

Editor de gráficos vectoriales que está destinado a la creación artística de dibujo y pintura para ilustración.

Gimp

Programa de edición de imágenes digitales en forma de mapa de bits, tanto dibujos como fotografías.
